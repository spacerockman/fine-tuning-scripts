# Bookmark Health Manager - 书签健康管理器

一个现代化的 Chrome 扩展，自动扫描您的所有书签，检测无效链接，并提供美观的管理界面。

## ✨ 功能特性

- 🔍 **自动扫描** - 递归遍历所有书签文件夹
- ⚡ **快速检测** - 并发检查链接有效性（5秒超时）
- 🎨 **现代UI** - 渐变背景、玻璃态效果、流畅动画
- ✏️ **编辑功能** - 一键跳转到 Chrome 书签管理器
- 🗑️ **删除功能** - 快速清理无效书签
- 💾 **持久化存储** - 扫描结果自动保存

## 📦 安装方法

### 1. 下载扩展文件

将整个 `bookmark-manager` 文件夹保存到本地。

### 2. 在 Chrome 中加载扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 在右上角启用 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择 `bookmark-manager` 文件夹
6. 扩展将出现在工具栏中

## 🚀 使用方法

### 扫描书签

1. 点击工具栏中的扩展图标
2. 在弹出窗口中点击 **"开始扫描"** 按钮
3. 等待扫描完成（可能需要几分钟，取决于书签数量）
4. 查看检测到的无效书签列表

### 管理无效书签

每个无效书签卡片提供两个操作：

- **✏️ 编辑** - 在 Chrome 书签管理器中打开该书签进行编辑
- **🗑️ 删除** - 永久删除该书签（需要确认）

### 权限说明

- `bookmarks` - 读取和修改书签
- `storage` - 保存扫描结果
- `<all_urls>` - 检查链接有效性

### 链接检测机制

- 使用 `fetch()` API 发送 HEAD 请求
- 5秒超时限制
- `no-cors` 模式以支持跨域检测
- 网络错误或超时视为无效链接

## ⚠️ 注意事项

1. **误报可能性** - 某些网站可能阻止自动请求，导致误判为无效链接
2. **需要认证的网站** - 需要登录的书签可能被标记为无效
3. **扫描时间** - 大量书签可能需要较长扫描时间
4. **开发者模式** - 此扩展需要在开发者模式下加载

## 🐛 故障排除

### 扩展无法加载

- 确保已启用开发者模式
- 检查是否选择了正确的文件夹
- 查看控制台是否有错误信息

### 扫描无结果

- 检查是否有书签
- 查看浏览器控制台日志
- 尝试重新扫描

### 删除失败

- 确保书签仍然存在
- 检查扩展权限
- 重新加载扩展
